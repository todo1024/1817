课程介绍：

毋庸置疑，Python是DevOps领域首选的开发语言，因为它易学易用，能够满足大部分自动化需求，快速开发出高大上的运维管理平台，是目前系统运维应用最广泛的语言，没有之一！

但由于运维人员没有太多编程经验，就导致了很多初学者无从下手，无法开发运维平台，很难真正的“掌握”！
